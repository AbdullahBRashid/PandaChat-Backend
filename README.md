## Usage

    npm install

#### Development Mode
The development mode runs the TypeScript Compiler with `--watch=true` and the Nodemon service for reload on save.

    npm run dev

#### Build
Runs the `tsc` command to build files into directory `dist/`.

    npm run build

#### Starting
Runs the node server

    npm run start
